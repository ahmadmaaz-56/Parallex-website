# Parallex-website
Parallax Scrolling Website A modern, visually engaging website that demonstrates the popular parallax scrolling effect where background images move slower than foreground content, creating an immersive 3D-like experience.
Parallax Scrolling Website
A beautiful demonstration of parallax scrolling effects using pure CSS and HTML.
Features
-  Pure CSS parallax scrolling effect
-  Fully responsive design
- High-quality background images
-  Clean, modern UI with overlay content
-  No JavaScript required
-  Educational implementation

Technologies Used

- HTML5
- CSS3 (background-attachment, vh units)
- Responsive Design
- Unsplash API for images

 How It Works
The parallax effect is achieved using CSS `background-attachment: fixed` which creates the illusion of depth by moving background images slower than the foreground content during scrolling.
